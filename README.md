# Online-Voting-System-WebApp-Python
**Online Voting System Web-App in python.**

**Libraries Used :**
  1. Pywebio 
  2. MySql-Connector

**Features :**
  1. SignUp & Login.
  2. Only user having age 18 or above can vote.
  3. User can vote only once.
  4. User can check results immediately after voting.
